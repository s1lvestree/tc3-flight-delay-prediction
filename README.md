# Flight Delay Prediction

Projeto de Machine Learning para prever atrasos em voos.

## Objetivo

Construir um modelo capaz de prever se um voo irá atrasar com base em características operacionais.

## Dataset

Dataset de voos contendo informações como:

- companhia aérea
- aeroporto de origem
- aeroporto de destino
- horário de partida
- distância do voo
- atraso na chegada

## Etapas do projeto

1. Exploração de dados (EDA)
2. Engenharia de features
3. Modelagem supervisionada
4. Avaliação de modelos

## Modelos utilizados

- Logistic Regression
- Random Forest

## Métricas avaliadas

- Accuracy
- Precision
- Recall
- F1 Score
- ROC-AUC

## Resultados

| Modelo | ROC-AUC |
|------|------|
Logistic Regression | 0.71 |
Random Forest | 0.76 |
